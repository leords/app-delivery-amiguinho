Amiguinho Delivery

O Amiguinho Delivery é um aplicativo desenvolvido para otimizar o processo de conferência e entrega de pedidos do sistema Amiguinho.

O aplicativo permite que o entregador visualize os pedidos carregados para sua rota, realize a conferência das mercadorias e acompanhe todo o fluxo da entrega até a conclusão no cliente.

Durante a finalização da entrega, o sistema registra automaticamente:

Horário da entrega;
Localização (GPS) do dispositivo;
Status da entrega.

A autenticação é realizada utilizando as credenciais do próprio sistema Amiguinho, garantindo que apenas usuários autorizados tenham acesso às funcionalidades do aplicativo.

Tecnologias utilizadas
React Native
Expo
JavaScript
API REST do sistema Amiguinho
Geolocalização (GPS)
AsyncStorage para persistência local de dados
React Navigation
Principais funcionalidades
Autenticação integrada ao sistema Amiguinho;
Listagem de pedidos carregados para entrega;
Conferência dos pedidos;
Atualização do status das entregas;
Registro da localização GPS na conclusão da entrega;
Registro automático do horário de entrega;
Atualização das informações em tempo real através da API do sistema.
